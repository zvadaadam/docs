<Tabs>
<Tab title="Web">
  - Go to <a href="https://steercode.com" target="_blank" rel="noreferrer">steercode.com</a>  
  - Sign in and create a new project  
  - Describe your idea and generate
</Tab>

<Tab title="iOS">
  - Install the app from the <a href="https://apps.apple.com/us/app/id6742994386" target="_blank" rel="noreferrer">App Store</a>  
  - Sign in with the same account  
  - Create a project and generate on the go
</Tab>
