### How it works

1. Open <a href="https://steercode.com" target="_blank" rel="noreferrer">steercode.com</a> (or the iOS app)  
2. Describe the website or app you want  
3. Click **Generate** and iterate via chat

<Tip>
If you plan to collect data or handle auth, mention it. SteerCode can scaffold **Supabase** auth, database tables, and typical CRUD screens. See <a href="/how-to/integrations">Integrations</a>.
</Tip>
